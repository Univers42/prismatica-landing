# table/

🚧 **placeholder** — future interactive data table view layout.

reserved for a full-featured data table — the upgraded version of the simple `Table` atom.

## what it will probably include

- sortable columns
- filtering and search
- row selection
- pagination or virtual scrolling
- column resizing and reordering
- inline cell editing

## how it differs from the Table atom

the `Table` atom is a simple static table for displaying data. this `table/` view would be an interactive data grid with user controls.
