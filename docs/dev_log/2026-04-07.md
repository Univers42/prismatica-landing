### 🎯 Objective
Establish the foundational infrastructure for a private user dashboard area, completely decoupled from the landing page's immersive aesthetic, following strict FSD principles and secure routing.

### ✅ Key Achievements
- **Routing & Protection**:
    - Implemented a **`ProtectedRoute`** gate component that consumes the Zustand auth state.
    - Configured a lazy-loaded `/dashboard` route in `AppRouter.tsx`.

    - Created a professional **Sidebar** layout with navigation items and user profile info.
